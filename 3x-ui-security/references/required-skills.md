# Required companion skill

Для запросов о сокрытии VPN-трафика, обходе DPI, active probing или выборе
маскировки дополнительно требуется `nidox-vpn-detection-defense-skill`.

Если companion skill недоступен, не имитировать его вывод: явно сообщить об
ограничении и продолжить только с документированными возможностями 3X-UI/Xray.
